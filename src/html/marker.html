NetChart.settings.legend.marker

Net Chart NetChart Settings legend marker
This section is defined by the type BaseSettingsLegendMarker.

Visual element of legend entry with appropriate style to a slice color it corresponds. The content of each legend marker is the same as info popup appearing while hovering on slice.

Properties

alignment: "left" | "right"
Default value: "left"

Marker position relative to text

Valid values:
  • left
  • right
lineColor: string
Default value: "#fff"

Line color around marker shape

size: number
Default value: 30

Marker size.
